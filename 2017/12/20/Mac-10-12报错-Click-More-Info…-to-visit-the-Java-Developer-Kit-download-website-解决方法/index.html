Mac 10.12报错:Click More Info… to visit the Java Developer Kit download website.解决方法

Dec 20, 2017

  • 今天学习java，用vim写的正爽呢，弹窗报错如下图。根据报错升级了java，还是不行。上网查询，解决之.

  • 解决方法：

    1. 安装提示点击“More Info”先升级java版本
    2. 第一步安装的jre，如果你需要编译java，那么还需要升级jdk
    3. 安装完成jre，jdk问题就解决了
